What is a Cloud Server?

Cloud Server is a Data base that is hosted from a remote location. A customer buys a space from a company, who will host there webpage. The customer is unware of the location of their Server.


What makes this so fascinating?

A customers webpage can be hosted from across the world, before the servers where very large and had to be within a certain distance. The growth in technology field in recent years has made some many technologic advences.
